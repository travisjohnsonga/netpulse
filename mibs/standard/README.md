# Standard MIBs

RFC standard MIBs, freely available from the IETF/IANA. Drop the raw MIB files
here (`.my` / `.mib` / `.txt`).

Recommended set:
- `SNMPv2-SMI`, `SNMPv2-TC`, `SNMPv2-MIB`
- `RFC1213-MIB` (MIB-II)
- `IF-MIB` (interfaces)
- `IP-MIB`, `TCP-MIB`, `UDP-MIB`
- `HOST-RESOURCES-MIB`
- `ENTITY-MIB`

Source: https://www.rfc-editor.org / https://www.iana.org/assignments/mibs
